✂️ Simulador de Turnos - Peluquería

🧾 Descripción general
Este proyecto es un simulador interactivo de turnos para una peluquería, desarrollado como parte de la Entrega 1 del curso de JavaScript. El usuario puede elegir un servicio, seleccionar un horario disponible y confirmar su turno. Toda la interacción se realiza desde la consola del navegador y mediante cuadros de diálogo (prompt, alert, confirm).

⚙️ Funcionalidades
- Mostrar un menú de servicios disponibles.
- Listar horarios disponibles.
- Reservar un turno si el horario está libre.
- Confirmar o cancelar la reserva del turno.
- Guardar y mostrar los turnos tomados en consola.
- Posibilidad de reservar múltiples turnos en una sola sesión.
